
# VCreative - Sitio Web Oficial

Sitio web profesional para VCreative, agencia creativa especializada en contenido para redes sociales.

## 🚀 Características

- ✨ Diseño moderno y responsivo
- 🎥 Carrusel de videos verticales (formato TikTok/Instagram)
- 🎨 Animaciones con Framer Motion
- 📱 Optimizado para móviles
- ⚡ Construido con Vite + React + TypeScript

## 🛠️ Tecnologías

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Shadcn/ui

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
